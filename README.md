# F.W.S.F.S.S
A repository for the Font Website Stolen Fonts and Suspicious System

Canva Project URL: https://canva.link/knyx8datj3cvjun

Font Website Stolen Fonts and Suspicious System (FWSFSS)

The goal of this system making it easier which websites should you use.

VERY IMPORTANT THING: Websites that allow users to upload fonts are moved up a rating in case if you where to upload a stolen font. Now, I know there might be moderation, but I put them in a higher rating just in case.

There are 5 base ratings with 4 in between. These are:

N - Minimal

NL - Minimal-Low

L - Low

LM - Low-Medium

M - Medium

MH - Medium-High

H - High

HX - High-Extreme

X - Extreme
